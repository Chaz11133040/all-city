How to use your font
====================

1. Copy the horizon_bold.css file and fonts/ folder to your website

2. Include the css file using the following link tag:

     <link rel="stylesheet" type="text/css" href="horizon_bold.css">

3. Reference the font in your CSS using font-family:horizon_bold

     Example:

     .mytext
     {
        font-family: horizon_bold;
        font-size:18px;
        color:#aaa;
     }

Important: Please read LEGAL.txt before using this font.
